# Model Provider Formats

The canonical runtime documentation is maintained with the coding-agent:

- [Model configuration](../../packages/coding-agent/docs/models.md)
- [Provider composition](../../packages/coding-agent/docs/providers.md)
- [Protocol format selection](../../packages/coding-agent/docs/model-provider-formats.md)

REPI ships protocol adapters, not provider/model presets. Configure a complete
`REPI_*` environment model, `~/.repi/agent/models.json`, or an extension. The
linked model guide is the single source for supported variables, input/output
and cache pricing, price tiers, context limits, headers, compatibility, and
thinking metadata.
