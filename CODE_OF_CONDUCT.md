# 行为准则

REPI Agent 社区以专业、可验证、尊重事实为基本原则。

## 我们鼓励

- 清晰描述问题、证据和复现步骤。
- 尊重不同经验水平的贡献者。
- 对代码、设计和结论提出具体、可验证的批评。
- 主动脱敏日志、截图和诊断材料。

## 不接受

- 人身攻击、骚扰、歧视、威胁或持续挑衅。
- 公开泄露他人的密钥、私有端点、目标信息或未脱敏日志。
- 批量提交低质量、不可复现或自动生成但无人审核的问题。
- 伪造测试结果、夸大能力或提交无法验证的声明。

## 执行

维护者可以编辑、隐藏或关闭违反准则的内容；严重或重复违反者可能被限制参与项目。安全敏感内容会优先转入私有处理流程。
